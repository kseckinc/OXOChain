# OXOChain

OXO Chain is an Ethereum-based blockchain. 
You can use it with an RPC connection with a wallet such as Metamask. 
OXO Chain uses the Proof-of-Authority consensus. 


# For Metamask:

RPC    : https://rpc.oxochain.com
Symbol : OXO
